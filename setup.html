Setup

This section describes how to configure your environment and run the AI credit scoring application locally. You will need AWS credentials, a MongoDB instance and the ability to run Python and Node.js applications.

Configure AWS and Environment Variables

Install the AWS CLI and authenticate if you use AWS Single Sign‑On: 

aws sso login --profile default
aws configure

Set the following environment variables before starting the backend:

# Basic credentials
AWS_ACCESS_KEY_ID=your-access-key
AWS_SECRET_ACCESS_KEY=your-secret-key
AWS_REGION=us-east-1

# Inference profile settings (choose one of ARN or ID)
BEDROCK_TEXT_REGION=us-west-2
BEDROCK_TEXT_INFERENCE_PROFILE_ARN=arn:aws:bedrock:us-west-2:ACCOUNT_ID:inference-profile/my-text-profile
BEDROCK_TEXT_INFERENCE_PROFILE_ID=ip-1234567890abcdef
BEDROCK_EMBED_REGION=us-west-2
BEDROCK_EMBED_INFERENCE_PROFILE_ARN=arn:aws:bedrock:us-west-2:ACCOUNT_ID:inference-profile/my-embed-profile

# On‑demand model settings (omit model ID when using an inference profile)
BEDROCK_TEXT_MODEL_ID=anthropic.claude-v2
BEDROCK_EMBED_MODEL_ID=amazon.titan-embed-text-v1

# Optional external services
FRAUD_DETECTOR_MODEL_ARN=arn:aws:frauddetector:REGION:ACCOUNT_ID:detector/my-detector
SAGEMAKER_ENDPOINT_NAME=my-anomaly-endpoint

# Database connection
MONGODB_URI=mongodb://localhost:27017

Run the Backend

To start the API server, install dependencies and run the FastAPI app:

cd backend
# create a .env file containing the variables above
pip install -r requirements.txt
uvicorn main:app --reload

Run the Frontend

To launch the React front‑end, run the following commands:

cd frontend
npm install
npm run dev   # start the development server
npm run build # create a production build
npm start     # preview the production build

Once both services are running, open http://localhost:5173/ in your browser to interact with the application.
